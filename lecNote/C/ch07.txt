문자열 str은 문자열의 주소를 가르키고 str을 가르키는 주소도 따로 있다 
*pointer = &a 일 때 *pointer의 값을 바꾸면 a값이 바뀐다
포인터로 배열의 맨앞자리 주소를 받으면 배열과 똑같이 사용가능하다
*(arr+2)는 arr[2] 와 같다
call by value함수에선 변수의 값만 받아와서 값에 대한 새로운 주소를 부여하고 이에 대해 연산을 수행함
swap 함수를 사용하려면 call by refernce 함수를 사용해야한다
함수의 매개변수로 배열 대신 포인터를 받으면 모든 크기의 배열을 받을 수 있다
함수의 return값이 배열일 수는 없고 포인터를 return 해야한다 -> 함수 지정 시 int* 함수명(변수);
함수 내에서 사용한 변수가 메모리에서 지워지지 않기 위해서 static을 붙여준다
